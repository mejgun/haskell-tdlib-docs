data Update Source #

Contains notifications about data changes

Constructors

UpdateAuthorizationState

The user authorization state has changed

Fields

  • authorization_state :: Maybe AuthorizationState

    New authorization state

UpdateNewMessage

A new message was received; can also be an outgoing message

Fields

  • message :: Maybe Message

    The new message

UpdateMessageSendAcknowledged

A request to send a message has reached the Telegram server. This doesn't mean that the message will be sent successfully. This update is sent only if the option "use_quick_ack" is set to true. This update may be sent multiple times for the same message

Fields

  • chat_id :: Maybe Int

    The chat identifier of the sent message

  • message_id :: Maybe Int

    A temporary message identifier

UpdateMessageSendSucceeded

A message has been successfully sent

Fields

  • message :: Maybe Message

    The new message

  • old_message_id :: Maybe Int

    The previous temporary message identifier

UpdateMessageSendFailed

A message failed to send. Be aware that some messages being sent can be irrecoverably deleted, in which case updateDeleteMessages will be received instead of this update

Fields

  • message :: Maybe Message

    The new message

  • old_message_id :: Maybe Int

    The previous temporary message identifier

  • _error :: Maybe Error

    The cause of the message sending failure

UpdateMessageContent

The message content has changed

Fields

  • chat_id :: Maybe Int

    The chat identifier of the sent message

  • message_id :: Maybe Int

    A temporary message identifier

  • new_content :: Maybe MessageContent

    New message content

UpdateMessageEdited

A message was edited. Changes in the message content will come in a separate updateMessageContent

Fields

  • chat_id :: Maybe Int

    The chat identifier of the sent message

  • message_id :: Maybe Int

    A temporary message identifier

  • edit_date :: Maybe Int

    Point in time (Unix timestamp) when the message was edited

  • reply_markup :: Maybe ReplyMarkup

    New message reply markup; may be null

UpdateMessageIsPinned

Fields
